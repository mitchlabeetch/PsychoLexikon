# PsychoLexicon

Ein interaktives Psychologie-Lernportal für Studierende im ersten Semester. Zwölf Kernthemen des Bachelor-Studiums, erklärt, veranschaulicht und zum Mitdenken gemacht.

## Canonical App

- Produktiver App-Code: `app/`
- Runtime-Content: `app/src/content/articles/*.json`
- Legacy-Migrationsquelle: `content/thema-*.yml`

## Wichtiger Hinweis

Die Verzeichnisse `app_archive/`, `app_new/`, `project/` und `project2/` sind keine kanonischen Laufzeitziele. Die aktuelle Content-Architektur, Migration und Validierung sind in `app/README.md` dokumentiert.
